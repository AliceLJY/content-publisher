### ai-custom (#17)

When this theme is selected in rotation, the AI analyzes the article's tone, topic, and mood, then generates a one-time custom CSS theme. This theme is used once and not saved for reuse.

## Auto-Rotation Logic

1. Read `~/.openclaw-antigravity/workspace/images/layout-style-history.txt`
2. Find last used theme number
3. Next = last + 1 (wraps to 1 after 17)
4. Check tone match against article category/mood
5. If mismatch, skip to next
6. Pass `--theme <theme-key>` to the publish command

### History File Format

```
YYYY-MM-DD article-slug #NN theme-key display-name
```

Example:
```
2026-02-28 ai-agent-workflow #11 wechat-anthropic Claude
2026-03-01 wechat-publishing #12 xiaomuwu-journal Xiaomuwu Journal
```

## CSS Theme Generation

Custom theme CSS files are generated by `scripts/generate-layout-themes.ts`:

```bash
bun scripts/generate-layout-themes.ts
```

This script:
1. Reads the STYLES object from the editor HTML (`wechat_editor_xiaoshiai.html`)
2. Converts inline style definitions to CSS files
3. Writes them to `scripts/themes/`
4. Skips default/grace/simple (reserved names)
5. Generates `layout-style-catalog.md` at repo root

## Inline Formatting

`scripts/format-wechat.ts` applies inline styles directly to HTML elements (WeChat strips `<style>` tags). It:
- Parses frontmatter for title/author/summary
- Renders markdown via markdown-it with highlight.js
- Applies the selected theme's inline styles to every element
- Outputs a complete HTML document ready for WeChat
